# Redfetch

A tiny red terminal system fetch with a cleaner ASCII badge.

```powershell
go run .
```

Build:

```powershell
go build -o redfetch.exe .
.\redfetch.exe
```

Shows:

- user/host
- OS + arch
- CPU
- RAM
- all mounted disks
- LAN IP
- uptime

No external Go dependencies. One file, one binary.
